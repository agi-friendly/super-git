
# Tortoise GIT
- 👍👍👍 Windows 탐색기의 컨텍스트 메뉴에서 바로 사용 가능
- 👍👍 Amend Commit 시 이전 커밋의 파일 제외 가능
- Git Worktree 지원 (살짝 구림)
  - 일괄 제거 가능
- 👍 Git Rebase
  - 실패 없음 (IntelliJ 는 도중에 실패 시 전체 실패)
  - 리베이스 전 중간단계 제공으로 커밋 별 Skip, 메세지 변경 편함
  - 동일한 변경 자동 Skip 처리
- Show Log
  - 👍 로컬 변경사항 Stash 커밋이 아주 편함
  - 탐색기 기반이라, 같은 저장소를 여러 로그창을 띄워서 비교 가능
  - 👎 한글 검색 타이핑 힘듬(클립보드 붙여넣기 필요)
  - 👎 검색 느림
- 👍 2 File Diff 지원
  - 어디서든 2개 파일 비교 가능
- 👍 가벼움!!!
- 👍 Git Reflog 지원
- 예전 커밋 기준 저장소 탐색 (Browse Repository)
- 👍 Push 시 기본 원격 저장소 설정 가능
- 이전 커밋 메세지 불러오기
- 브렌치 관리하기 유리함 (다양한 정렬)
  - 브렌치 이름별, 날짜별, 마지막 Author 별, 마지막 커밋 시간별
  - `/` 기준으로 tree depth 표현
  - 로컬 브렌치는 Traked branch 설정으로, 따르는 원격 브렌치 변경이 가능
- 커밋 기준 zip 파일 생성 가능

# Git Worktree Manager (vscode)
- 👍👍 Git 저장소 즐겨찾기
- 워크트리 관련 기능 아주 편함
- 👍 Worktree 생성 시 패턴 생성 지원
- 각 워크트리 별 즉시 해당 vscode 열기, 즉시 터미널 실행 가능

# IntelliJ IDEA - GIT
- ⚠️ Git Rebase
  - 동일한 변경 자동 Skip 처리
  - 실패 시 전체 실패
- 👍 Conflict 처리
  - IDE 내에서 동작으로, 컴파일 오류 즉시 확인 가능
    - Tortoise Git으로 리베이스 수행하고, 충돌 시 IntelliJ IDEA 에서 충돌 처리 가능
- 👍 중간 커밋 메세지 변경 가능
- 👍 Interactively Rebase
  - 중간 커밋 병합 가능
  - 중간 커밋 순서 변경 가능
  - 중간 커밋 메세지 변경 가능
  - 중간 커밋 제거 가능
- Shelve 기능 지원
- 👍 3 File Diff 지원
- 중간 커밋 제거 가능 (Drop Commit)
- 마지막 커밋 취소 가능 (Undo Commit)
- 예전 커밋 기준 저장소 탐색 (Show Repository at Revision)
- 이전 커밋 메세지 불러오기
- 👍 로컬 변경사항 기반으로 AI가 커밋 메세지 생성 가능
- 👍 Create Patch, Apply Patch 기능 지원
- 👍 클립보드로 Patch 복사, 붙여넣기 가능
- 커밋 별 AI 리뷰 가능
